# ProyectoFinalEstDatos
Portafolio

* **[ADA 01 - Pilas: Parser de Expresiones Aritméticas](./ADA01)**
    * **Descripción breve:** Implementación de un analizador sintáctico (parser) que lee expresiones matemáticas en notación infija desde un archivo, las convierte a notación postfija (Polaca Inversa) utilizando una pila dinámica y evalúa el resultado final.

* **[ADA 02 - Colas: Simulador de Atención al Cliente](./ADA02)**
    * **Descripción breve:** Desarrollo de un simulador de atención al cliente en un banco que utiliza tres colas (Queues) separadas, implementadas con nodos enlazados en C, para calcular estadísticas clave como tiempos de espera promedio y eficiencia de servicio.

* **[ADA 03 - Listas Dobles: Gestor de Catálogo de Películas](./ADA03)**
    * **Descripción breve:** Implementación de una Lista Doblemente Ligada en C para cargar, administrar y manipular registros de películas desde un archivo CSV. Incluye funciones CRUD (Crear, Leer, Actualizar, Eliminar) y diversas operaciones de búsqueda.

* **[ADA 04 - Árboles Binarios: Gestor de Estudiantes](./ADA04)**
    * **Descripción breve:** Implementación de un Árbol Binario de Búsqueda (ABB) en C para gestionar registros de estudiantes (matrícula, nombre, calificaciones dinámicas). El sistema permite inserciones, eliminaciones, reportes ordenados y búsquedas basadas en la matrícula.
